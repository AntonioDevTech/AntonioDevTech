### **🛡️ [Project Titan: Autonomous Cloud Infrastructure](https://alfanotechport.org)**
**Status:** *Live Production System* | **Role:** *Lead Architect*

A self-healing hybrid cloud infrastructure designed to survive active network attacks without human intervention.
* **The Architecture:** Architected a live, self-healing cloud ecosystem on **Microsoft Azure** using **Terraform (IaC)**. Decoupled software from physical hardware to ensure 100% data persistence via redundant SQL backups.

### **⚡ [Miner Control Center (MCC)](https://github.com/AntonioDevTech)**
**Status:** *v1.0 Released* | **Role:** *Sole Developer*

A high-frequency desktop telemetry engine that unifies mixed fleets of **10,000 ASICs** into a single pane of glass.
* **Core Engineering:** Bypasses standard slow HTTP interfaces to establish **Raw TCP Socket** connections (Port 4028) directly to miner firmware.
* **Universal Translation:** Custom engine standardizes unstructured string data from Antminers and JSON streams from Whatsminers into a unified C# object model.
* **Thermal Automation:** Asynchronous "Watchdog" thread polls thermal sensors and triggers emergency shutdowns if hashboards exceed 85°C to prevent silicon failure.
* **Stack:** `C# .NET 8` `TCP/IP` `WPF` `SQLite` `Regex`

### **🔧 Tier 3 Hardware Operations**
**Status:** *Active Skillset* | **Role:** *Hardware Engineer*

Deep component-level diagnostics and repair for high-performance computing hardware.
* **Diagnostics:** Use Fluke multimeters and oscilloscopes to trace voltage rail failures (1.8V, 0.8V, 12V) on control boards, identifying shorted MOSFETs and dead LDO regulators.
* **Physical Layer:** Crimp custom ethernet cables (CAT5e to CAT8) and diagnose packet loss at the physical layer.
